# 第5章 宽带LFMCW诊断系统验证与等效色散靶标时延轨迹提取

# 5.1 引言

第五章的目标是在受控条件下，对前文建立的色散特征提取与参数估计链路进行工程化验证。由于真实等离子体在实验室环境中难以长期稳定复现，其瞬态波动容易干扰对算法本身性能的判断，因此本章引入结构参数已知、微波响应可稳定获取的带通滤波器，作为等效色散验证载体。

本章的验证工作沿两条主线展开。第一条主线面向系统硬件分辨能力，构建宽带LFMCW收发前端并完成空间位移条件下的时延分辨率标定，用于评估系统对皮秒级时延变化的跟踪能力（5.3节）；第二条主线面向算法链路验证，引入等效色散载体（5.4节），在全链路仿真环境中完成差频时延特征提取与已知模型条件下的参数一致性检查（5.5节）。两条主线分别对应硬件极限与算法效能的验证，共同为本文方法在受控条件下的工程可行性提供依据。

# 5.2 宽带LFMCW诊断系统设计与时延分辨率测试

第三章和第四章的推导表明，低电子密度状态的诊断下限受制于系统对微小时延波动的分辨能力，而该物理极限直接依赖射频带宽。本节据此完成硬件链路拓扑设计、非色散环境去嵌入，并开展移动靶标时延分辨率标定。系统核心参数配置为：工作频段覆盖Ka波段，主扫频带宽扩展至3 GHz，测量误差控制在15%以内。

## 5.2.1 宽带LFMCW收发前端架构与扩频链路设计

诊断系统采用超外差架构完成基带扫频信号到Ka波段宽带信号的频率搬移。直接在毫米波频段产生宽带线性调频信号具备较高的硬件代价，故本系统将线性度控制集中在低频段，再通过混频与倍频链路逐级搬移到目标频段。

前端射频链路采用分级外差与倍频相结合的结构完成宽带Ka波段信号生成。基带扫频信号先与低频本振混频产生中频，再经级联二倍频链路提升至13.2~14 GHz波段，最后与21 GHz本振再次混频，得到覆盖Ka频段的发射信号。接收端通过功分保留参考支路，并与探测回波在解调器中混频，输出表征传播时延差异的差频信号。

该分级外差结构将频谱扩展分散到多级混频与倍频网络中，可降低单级高倍频带来的谐波和杂散压力，同时保留一定的频率配置灵活性。

LFMCW系统的固有时延分辨率满足$\Delta\tau = 1/B$。初始800 MHz配置对应1.25 ns的时延分辨尺度；若按前述诊断口径（载频$f_0 = 32$ GHz、$d = 200$ mm）进行量级换算，其对应的电子密度检出下限约为$1 \times 10^{18}$ m$^{-3}$。为提升对微小时延变化的跟踪能力，系统需要进一步扩展射频带宽。最终方案在Ka波段内实现3 GHz扫频带宽；其中硬件标定中心频率为34.6 GHz，强色散等效仿真工作频段选为37 GHz，二者均处于单扫全带覆盖范围内。

## 5.2.2 实验系统搭建与非色散环境基准测试

实验平台主要由扫频信号源、射频前端机箱、收发天线及采集处理模块组成。扫频信号由泰克AWG70001A产生，第一级与第二级本振分别由LMX2820评估板和E8267D提供。射频前端机箱内部集成混频、倍频、滤波和放大链路；收发端采用26~40 GHz宽带聚焦透镜天线，差频信号由高速示波器DSOX95004Q采集，并同步送入频谱仪监测。

平台搭建完成后，首先在空气信道中开展直通测试，用于标定系统在非色散环境下的本底响应。以800 MHz配置为例，发射信号覆盖34.2~35.0 GHz，差频信号经低通滤波后主要位于420 kHz~2 MHz范围，该频段主要由线缆长度和系统固有延迟决定。为减小常驻共模延迟对测量结果的影响，实验采用差分测量方式，仅以介入目标后差频的相对变化量$\Delta f_D$作为后续分析输入。

在扩频后的空传测试中，差频频移随扫频带宽增加而呈现稳定增长趋势，与$f_D = B\tau/T_m$的理论关系一致。3 GHz配置下的实测发射频谱如图5-2所示，在33.1~36.1 GHz范围内实现连续覆盖，$-3$ dB带宽达到3 GHz。虽然带内仍存在约3~5 dB起伏，但未出现明显断裂，说明扩频后链路能够保持基本完整的宽带传输特性。

## 5.2.3 移动靶标时延测量与硬件系统极限分辨率标定

系统极限分辨能力的实验标定，依赖单轴高精度位移平台引入受控传播时延增量。以初始天线间距为参考零位提取相对差频，随后沿轴向移动接收天线$\Delta d$并再次采集差频信号。通过差分运算可抵消环境中的常驻杂散，得到由位移引起的频率增量$\Delta f_D$，并按式(5-1)计算测量时延增量

$$\Delta\tau_{meas} = \frac{\Delta f_D \cdot T_m}{B} \tag{5-1}$$

在自由空间条件下，其对应的理论时延增量为

$$\Delta\tau_{theory} = \frac{\Delta d}{c} \tag{5-2}$$

进一步定义相对误差率为

$$\varepsilon = \frac{|\Delta\tau_{meas} - \Delta\tau_{theory}|}{\Delta\tau_{theory}} \times 100\% \tag{5-3}$$

为评估系统在基础带宽配置下的分辨能力，实验首先在800 MHz带宽条件下进行标定，并引入LNA以改善回波信噪比。测试以平均相对误差不超过15%、且五次重复采集结果保持集中为判据，对10 mm至5 mm位移进行步进测量。结果表明，系统能够稳定分辨6 mm位移对应的约20 ps理论时延增量；当位移进一步减小至5 mm时，重复测量结果开始明显发散，已难以满足稳定分辨要求。据此，可将800 MHz配置下的有效时延分辨率标定为20 ps（对应空间位移约6 mm）。若按前述诊断口径进行量级换算，其对应的电子密度检出下限约为$1 \times 10^{18}$ m$^{-3}$。

为验证扩频体制对分辨能力的提升作用，系统进一步在1.6 GHz、2.4 GHz和3 GHz配置下开展位移标定实验。表5-3汇总了不同带宽下的测试结果。

# 5.3 微波带通滤波器的色散物理等效机理

5.3节的标定表明，LFMCW系统在非色散环境下已具备皮秒级时延分辨能力。下一步不再直接进入真实等离子体实验，而是先引入一个色散函数形式已知、响应真值可稳定获得的受控载体，用于验证第五章的方法链路能否可靠提取时延轨迹特征，并检验这些特征在已知正向模型条件下是否足以支撑后续参数估计。基于这一目的，本文选用微波带通滤波器作为等效验证载体，其中心频率、绝对带宽和原型阶数等设计参数明确，$S$参数与群时延可通过仿真稳定导出，适合构造受控色散场景并提供真值参考。

## 5.3.1 滤波器色散演化与等离子体截止谐振的物理同构性

色散介质中群时延的显著变化通常集中在特征频率附近。对于Drude等离子体，第三章已给出其相对群时延与截止频率$f_p$之间的关系：

$$\Delta\tau_g(f) = \frac{d}{c}\left(\frac{1}{\sqrt{1-(f_p/f)^2}} - 1\right), \quad f > f_p \tag{5-4}$$

当$f$从高频端逼近$f_p$时，群时延迅速增大，说明截止频率附近是色散信息最集中的区域。

带通滤波器虽然不具备等离子体的单边截止结构，但在通带上下边缘同样存在由"可通过"向"被抑制"的快速过渡。对于中心频率为$F_0$、绝对带宽为$BW$的滤波器，其上下边缘$f_L = F_0 - BW/2$和$f_H = F_0 + BW/2$附近都会出现相位快速变化，从而在群时延曲线上形成明显峰值。就局部观测特征而言，这种边缘色散响应与等离子体在截止频率附近的强色散行为具有相似性。

因此，在本章的等效验证中，重点并不在于滤波器与等离子体在全局曲线形态上完全一致，而在于二者都能在特征频率附近产生显著的群时延轨迹，并由少数主导参数控制其主要拓扑结构。滤波器的工程优势在于：其响应真值可由仿真或网络测量稳定获得，结构参数明确，且不受等离子体瞬态波动影响，适合作为受控的色散验证载体。

## 5.3.2 基于切比雪夫传递函数的群时延正向理论模型构建

为检验提取得到的时延特征散点是否具备足够的信息承载能力，需要预先建立与目标滤波器结构相对应的正向群时延模型。本文选用经典切比雪夫（Chebyshev）Type-I带通滤波器作为等效解析靶标，其群时延曲线主要由中心频率$F_0$、绝对带宽$BW$、原型阶数$N$以及通带纹波$R_p$等参数决定。

在微波网络理论中，群时延定义为传输相位对角频率导数的负值，即$\tau_g(\omega) = -d\phi(\omega)/d\omega$。对于N阶切比雪夫滤波器，其低通原型到带通结构的频率映射以及复传递函数构建均属于标准过程，本文不再展开基础推导，仅保留数值实现后的群时延模型$\tau_g^{theory}(f; F_0, BW, N, R_p)$，作为后续验证中的理论真值。

在标称参数$(F_0 = 37$ GHz，$BW = 1$ GHz，$N = 5$，$R_p = 0.5$ dB$)$下，模型可产生典型的双峰群时延结构：通带上下边缘附近分别出现峰值，通带中央则形成较平坦的时延基线。该模型能够为后续的特征提取误差分析和参数一致性检查提供稳定的参考曲线。

## 5.3.3 前向物理模型参数敏感度分析

在建立正向模型后，还需分析各参数对群时延拓扑结构的控制作用，以判断哪些参数在后续拟合中具有较高可观测性。由于纹波深度$R_p$较小，其主要影响通带内的细微波动，因此此处主要考察$F_0$、$BW$和$N$三项。

以基准参数$F_0 = 37$ GHz、$BW = 1$ GHz、$N = 5$为例，图5-3展示了群时延曲线随不同参数变化的定性演化结果。结果表明：中心频率$F_0$主要控制双峰在频率轴上的整体位置；带宽$BW$主要控制双峰间距及峰值结构的横向展开范围；而阶数$N$更多影响边缘陡度和局部纹波细节。

因此，从可观测性角度看，群时延主拓扑对$F_0$和$BW$最为敏感，而对$N$的响应相对较弱。基于这一点，后续一致性检查中固定$N=5$，仅对决定主拓扑位置和尺度的$F_0$与$BW$进行联合估计。

# 5.4 色散等效介质的时延轨迹提取与物理映射机理

5.4节已经确立了微波带通滤波器作为等效验证载体的作用。本节将在此基础上检验两个核心问题：LFMCW差频信号中能否稳定提取色散介质的时延轨迹特征点，以及在正向模型已知的条件下，这些特征点是否足以驱动后续参数估计。只要这两个问题得到肯定回答，前文形成的特征提取与估计思路便可自洽地从滤波器场景泛化到Drude等离子体场景。

围绕这一目标，本节主要完成以下论证：在ADS中搭建与前文硬件链路保持一致的全链路仿真平台，并通过去嵌入剥离系统固有延迟；建立从滑动窗口差频估计到频率-时延散点的物理映射关系，并结合物理约束完成数据清洗；在已知切比雪夫正向模型条件下进行MCMC一致性检查，评估散点集合的信息承载能力与后验收敛性。

## 5.4.1 全链路联合仿真与色散基准去嵌入

为在受控条件下复现5.3节硬件系统的工作方式，本文在ADS中建立了与实际射频前端相对应的电路级瞬态模型。信号流程包括基带扫频、初级变频、三级二倍频、二次上变频、功分以及自混频解调。仿真扫频范围设为$f_{start} = 34.4$ GHz至$f_{end} = 37.61$ GHz，工作频段覆盖目标滤波器的色散区间。瞬态仿真采用自适应变步长时间网格，最小步长5 fs，最大步长0.5 ps，总时长$T_{stop} = 0.55~\mu$s，以保证至少覆盖一个完整扫频周期。

目标色散介质选用5阶切比雪夫Type-I带通滤波器BPF11，其参数与5.4节正向模型保持一致：$F_0 = 37$ GHz，$BW = 1$ GHz，$R_p = 0.5$ dB，阻带衰减大于90 dB。滤波器置于功分后的测量通路中，用于模拟色散介质对LFMCW信号的群时延调制。图5-5给出了ADS全链路拓扑，其结构与实际射频前端保持一致，是后续信号分析与去嵌入处理的基础。

![图5-5 LFMCW诊断系统ADS电路级全链路仿真拓扑架构](../writing/figures/final_output_doc/fig5.5.png)

进一步地，还需对系统固有延迟进行去嵌入标定。差频信号中不仅包含目标滤波器引入的群时延，也叠加了前端走线、功分器、衰减器与混频器带来的系统固定延迟$\tau_{sys}$。若不先剥离这一部分，后续得到的时延散点会整体偏移，尤其会影响通带平坦区的精度评价。

去嵌入采用直通参考测量实现。在不接入目标滤波器时，差频信号仅由系统固有延迟决定。根据LFMCW差频与时延的基本关系$f_D = B\tau/T_m$，可由参考条件下的差频峰值频率$f_{D,thru}$反推出系统基准延迟：

$$\tau_{sys} = \frac{f_{D,thru} \cdot T_m}{B} \tag{5-5}$$

为避免多级混频和倍频链路中的互调杂散影响峰值搜索，频谱搜索窗限定在0~5 MHz范围内，并结合汉宁窗FFT与三角插值进行精细估计。最终得到系统基准延迟为

$$\tau_{sys} = 0.2470\ \text{ns} \tag{5-6}$$

后续所有含滤波器的群时延估计均以该值作为去嵌入校正常数。

## 5.4.2 离散时延特征点与连续演化曲线的物理映射关系

滑动窗口方法的物理含义，是将时间轴上的局部差频估计映射回扫频过程中的瞬时探测频率。LFMCW发射信号的瞬时频率如式(5-7)所示

$$f_{tx}(t) = f_{start} + K t \tag{5-7}$$

其中$K$为扫频斜率。以窗口中心时刻$t_c$对应的差频估计值$f_{IF}(t_c)$为例，该频率对应的探测频率为式(5-8)

$$f_{probe}(t_c) = f_{start} + K t_c \tag{5-8}$$

在该频率点，差频与群时延满足式(5-9)

$$f_{IF}(t_c) = K[\tau_{sys} + \tau_g(f_{probe})] \tag{5-9}$$

因而去嵌入后的群时延估计可写为式(5-10)

$$\hat{\tau}_g(f_{probe}) = \frac{f_{IF}(t_c)}{K} - \tau_{sys} \tag{5-10}$$

随着滑动窗口沿时间轴推进，便可获得一组离散观测点$\{(f_k, \hat{\tau}_k)\}$。这组散点在通带平坦区应聚集于约2 ns的基线附近，在通带边缘则向双峰区上升。若散点能够连续覆盖这一演化过程，说明LFMCW差频信号已经承载了色散曲线的主拓扑信息。

## 5.4.3 强色散时延轨迹特征提取与物理约束清洗策略

ADS输出的中频信号采用自适应变步长网格，因此首先需要重采样到均匀时间轴。本文以4 GHz采样率进行样条插值重采样，再通过4阶Butterworth低通滤波器保留200 MHz以内的差频主分量，并做2倍抽取，最终工作采样率为2 GHz。在预处理后的差频信号上，采用第四章建立的滑动窗口ESPRIT方法提取频率—时延散点。窗口长度设为总信号长度的3%，且不少于64个采样点；步进长度取窗口长度的$1/8$；子空间维度$L_{sub}$取窗口长度的$1/2$；信号源数由MDL准则自动判定，上限设为3。每个窗口输出瞬时差频$\hat{f}_{IF}$、对应探测频率$f_{probe}$和局部RMS幅度$A_{rms}$。

第一层清洗采用基于幅度的自适应阈值：

$$A_{rms}(f_k) > 0.20 \times \max\{A_{rms}\} \tag{5-11}$$

该阈值对应约-14 dB的相对幅度门限，用于剔除阻带边缘因信号严重衰减而产生的不稳定估计点。

第二层清洗采用四分位距门限：

$$\tau_{lb} = Q_{25}(\mathcal{T}_{core}) - 1.5\,\mathrm{IQR}(\mathcal{T}_{core}) \tag{5-12}$$

其中1.5倍IQR系数采用统计学中常用的异常值识别经验规则，用于进一步剔除极少数非物理离群点。

$$\hat{\tau}_g(f_k) > \tau_{lb} \tag{5-13}$$

由于锯齿扫频在斜率折返点附近会引入明显的反相交叠失真，因此按式(5-14)定义扫频边界保护区，并直接剔除位于保护区内的散点：

$$\delta_t = \max\left(0.01, \frac{T_w}{2T_m}\right), \qquad \delta_t < t_c/T_m < 1 - \delta_t \tag{5-14}$$

当前参数下$\delta_t \approx 0.029$，对应约2.9%的边界保护区。经检查，该步骤仅额外剔除了少量失锁点，保留下来的均为原始观测散点。

经去嵌入和三步清洗后，最终保留45个有效散点，频率范围为36.43~37.50 GHz，群时延范围为1.10~7.59 ns，完整覆盖双峰演化区间。图5-10给出了清洗后散点与理论真值曲线的对比结果。

![图5-10 经去嵌入与物理约束清洗后的群时延特征散点与ADS理论真值对比](../writing/figures/final_output_doc/fig5.11.png)

为定量的评价特征轨迹提取精度，将45个有效散点与ADS导出的群时延真值逐点比较。按局部色散梯度$|d\tau_g/df|$将频率轴划分为三个区间：通带平坦区对应$|d\tau_g/df| < 10$ ns/GHz的低梯度区域（36.7~37.3 GHz），色散双峰陡变区对应$|d\tau_g/df| > 50$ ns/GHz的高梯度区域（峰顶附近），色散过渡区为两者之间的中等梯度区域（过渡带）。统计结果见表5-5。

## 5.4.4 基于MCMC拟合的一致性检查与信息承载分析

为检验上述45个有效散点是否足以支撑已知正向模型下的主导参数估计，本节构建基于MCMC的一致性检查。按照5.4节的设计信息，固定原型阶数$N = 5$和纹波$R_p = 0.5$ dB，仅对决定主拓扑位置与尺度的中心频率$F_0$和绝对带宽$BW$进行采样。设观测数据为$D_{obs} = \{(f_k, \hat{\tau}_k)\}_{k=1}^{M}$，则后验分布满足

$$p(\theta | D_{obs}) \propto p(D_{obs} | \theta)\, p(\theta), \qquad \theta = [F_0, BW]^T \tag{5-15}$$

考虑到不同频段散点的信噪比存在差异，采用基于局部幅度的权重因子

$$w_k = \left(\frac{A_{rms}(f_k)}{\max\{A_{rms}\}}\right)^2 \tag{5-16}$$

并构建加权对数似然函数

$$\ln L(\theta) = -\frac{1}{2} \sum_{k=1}^{M} w_k \left(\frac{\hat{\tau}_k - \tau_g^{theory}(f_k; \theta)}{\sigma_{meas}}\right)^2 \tag{5-17}$$

其中$\sigma_{meas} = 0.2$ ns，用于表征观测噪声量级。为避免结果依赖过窄先验，参数先验采用宽泛的均匀分布：

$$F_0 \sim U(36, 38)\ \text{GHz}, \qquad BW \sim U(0.5, 2.0)\ \text{GHz} \tag{5-18}$$

采样采用四条独立初始化的Metropolis-Hastings链，每条链长度为18000步，前6000步作为预烧期丢弃。预烧期后每链保留12000个状态，总保留样本数为48000。最终$F_0$与$BW$的接受率均稳定在27%~29%区间。

收敛诊断结果显示，当前两参数一致性检查可得到$\hat{R}_{F_0}=1.0004$、$\hat{R}_{BW}=1.0002$，对应$ESS_{F_0}=5809$、$ESS_{BW}=5776$，积分自相关长度分别为8.26和8.31。图5-11所示的链轨迹与一维边缘分布在预烧期后进入相同高概率区域并保持良好重叠，说明当前采样已达到较稳定的收敛状态。

![图5-11 MCMC马尔科夫链游走轨迹与一维边缘后验分布](../writing/figures/final_output_doc/fig5.12.png)

后验统计的全面量化评价如表5-6所示。

因而，本章引入滤波器的意义在于提供一个色散函数形式已知、响应真值可控的验证载体，以检验LFMCW差频时延轨迹提取及模型一致性检查链路的有效性，而非将滤波器本身作为最终诊断对象。

# 5.5 本章小结

本章围绕前述理论与方法链路，在受控条件下完成了硬件标定与等效色散验证。实验引入结构参数已知、微波响应可稳定获取的切比雪夫带通滤波器，构建了一个不受真实等离子体瞬态波动影响的等效色散载体。

在硬件层面，本章构建了基于超外差架构的Ka波段宽带LFMCW测量系统，并通过空间位移标定实验验证了其在3 GHz带宽条件下具备约3.33 ps量级的有效时延分辨能力（5.3节）；在算法验证层面，基于ADS全链路仿真、系统延迟去嵌入与物理约束清洗，提取出了与理论真值总体一致的群时延散点轨迹（5.5节）。进一步地，在已知正向模型条件下，利用MCMC对主导参数$F_0$和$BW$进行一致性检查，得到与设计值接近的后验估计结果。

第五章的结果表明，在受控等效色散场景中，本文提出的时延轨迹提取方法能够从LFMCW差频信号中恢复主拓扑信息，并为后续的参数估计提供有效支撑。这为前文方法链在强色散诊断问题中的工程应用提供了验证依据。
