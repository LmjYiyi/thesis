# 4.5 Drude等离子体模型仿真验证与不确定性量化

第4.2节至第4.4节分别建立了降维前提、轨迹提取方法与贝叶斯反演框架。本节将这条"滑动窗口—MDL—ESPRIT—MCMC"链路落到Drude等离子体仿真场景中,回答两个直接问题:链路能否稳定提取频率-时延轨迹并完成电子密度反演;后验统计结果是否支撑4.2节提出的降维策略。

---

## 4.5.1 仿真环境设置与噪声建模

本节构建了完整的LFMCW等离子体诊断仿真系统,包含信号生成、Drude模型传播、噪声注入、混频处理和特征提取五个模块。表4-2汇总了仿真参数配置。

**表4-2 LFMCW等离子体诊断仿真参数配置**

| 参数类别 | 参数符号 | 物理含义 | 数值 | 单位 |
|---------|---------|---------|------|------|
| **LFMCW信号** | $f_{start}$ | 扫频起始频率 | 34.2 | GHz |
| | $f_{end}$ | 扫频终止频率 | 37.4 | GHz |
| | $B$ | 扫频带宽 | 3.2 | GHz |
| | $T_m$ | 调制周期 | 50 | $\mu$s |
| | $K$ | 调频斜率 | $6.4 \times 10^{13}$ | Hz/s |
| | $f_s$ | 仿真采样率 | 80 | GHz |
| **等离子体参数** | $f_p$ | 等离子体截止频率 | 33 | GHz |
| | $n_e$ | 电子密度（由$f_p$计算） | $1.3511 \times 10^{19}$ | m$^{-3}$ |
| | $\nu_e$ | 碰撞频率 | 1.5 | GHz |
| | $d$ | 等离子体层厚度 | 150 | mm |
| **传播路径** | $\tau_{fs}$ | 自由空间单程时延 | 1.75 | ns |
| | $\tau_{air}$ | 空气参考信道时延 | 4 | ns |
| **噪声模型** | SNR | 射频端信噪比 | 20 | dB |
| | $P_n$ | 噪声功率 | $P_s / 10^{2}$ | W |
| **ESPRIT参数** | $T_w$ | 滑动窗口时长 | 12 | $\mu$s |
| | 重叠率 | 窗口重叠比例 | 90 | % |
| **MCMC参数** | $N_{samples}$ | 总采样次数 | 10000 | 次 |
| | $N_{burn}$ | 预烧期 | 2000 | 次 |

上述参数确保探测频率$f \in [34.2, 37.4]$ GHz与截止频率$f_p = 33$ GHz之间满足$f > f_p$的透射条件,标准工况$f_p = 33$ GHz对应$(f_p/f)^2 \in [0.779, 0.931]$,覆盖从中等色散到强色散的宽动态范围。噪声注入采用射频端AWGN模型,功率由$P_n = P_s/10^{\text{SNR}_{\text{dB}}/10}$确定,注入位置选在混频之前的射频端口,符合实际接收机链路。设定SNR = 20 dB为标准条件,后续鲁棒性测试扫描10~30 dB。MCMC预烧期依据迹线图诊断选取$N_{burn} = 2000$。

信号传播采用频域精确仿真,等离子体层传递函数由Drude模型复介电常数导出:

$$H_{plasma}(\omega) = \exp\left[-j \cdot \text{Re}\{\tilde{k}(\omega)\} \cdot d - |\text{Im}\{\tilde{k}(\omega)\}| \cdot d\right] \tag{4-17}$$

其中复波数$\tilde{k}(\omega) = (\omega/c)\sqrt{\tilde{\varepsilon}_r(\omega)}$,复介电常数由式(3-1)给出。总传递函数包含两段自由空间固定时延与等离子体层色散与阻尼:$H_{total}(\omega) = H_{plasma}(\omega)\,e^{-j\omega(2\tau_{fs})}$。

## 4.5.2 特征提取精度及与传统方法的对比

本节从两个层面验证方法链的有效性:一是局部特征提取精度（ESPRIT vs 滑动窗口短时FFT）,二是整链路反演精度（本文方法 vs 传统全周期FFT单峰法）。

![图4-10 FFT与ESPRIT特征提取方法对比](figures/图4-9_FFT与ESPRIT对比.png)

ESPRIT散点紧密贴合理论真值曲线,而STFT脊线在低频端（接近截止频率处）表现出显著的系统性偏差。以标准工况（$f_p = 33$ GHz,SNR = 20 dB）为例,ESPRIT的RMSE为0.004 ns,STFT脊线的RMSE为0.110 ns,精度提升约30.1倍。该结果表明,轨迹提取层面的高精度是后续MCMC反演可信性的前提。

### 与传统全周期FFT单峰法的反演对比

为说明本章方法链在强色散诊断中的必要性,本节将"ESPRIT—MCMC"框架与传统全周期FFT单峰方法进行对比。两者的核心差异在于:传统方法假设差频信号为稳态单频正弦,采用全周期FFT峰值检测并通过线性近似公式反算电子密度,仅提供点估计且不具备不确定性量化能力,适用于弱色散区$(f_p/f)^2 < 0.5$;本文方法则将差频信号视为时变调频信号,通过滑动窗口—MDL—TLS-ESPRIT提取瞬时时延轨迹,再由基于完整Drude模型的MCMC贝叶斯推断完成反演,可覆盖$(f_p/f)^2 \in [0, 0.95]$的全色散范围,并提供后验分布、置信区间与CV判据。

设全周期FFT提取的单峰差频偏移为$\Delta f = f_{beat,plasma}^{corr} - f_{beat,air}^{corr}$,对应平均附加时延$\Delta\tau = \Delta f / K$。在弱色散线性近似下,传统方法所采用的电子密度点估计公式为:

$$\hat{n}_e^{FFT} \approx \frac{8\pi^2 \varepsilon_0 m_e c \bar{f}^2}{e^2 d}\frac{\Delta f}{K} \tag{4-18}$$

其中$\bar{f} = (f_{start}+f_{end})/2$为扫频中心频率。该公式将全周期FFT得到的单一差频峰值解释为全局平均时延,仅在弱色散、时延变化缓慢的条件下近似成立。表4-3汇总了不同截止频率下两种方法的电子密度反演结果（SNR = 20 dB）。其中传统方法的差频频率差定义为$\Delta f = f_{beat,plasma}^{corr} - f_{beat,air}^{corr}$（单位：MHz）,对应$\Delta\tau = \Delta f / K$。

# 4.6 本章小结

Drude仿真已证实:ESPRIT重构的频率—时延轨迹RMSE为0.004 ns,MCMC后验均值与真值的偏差在强色散主工作区不超过0.4%;48组参数扫描与10–30 dB的SNR扫描均不改变这一结论,$n_e$的后验CV始终维持在$1\%\sim6\%$。贴合这一数据趋势,后验角点图呈现$n_e$纵向尖峰、$\nu_e$横向弥散的特征结构,与4.2节“$\nu_e$仅为二阶微扰”的理论分析互相印证。与传统全周期FFT单峰法相比,后者在强色散区的电子密度误差可超过245%,而本文方法保持在不超过0.4%的水平。

基于上述定量结果,第四章的工程结论可归结为:在群时延主导的LFMCW诊断链路中,将$\nu_e$固定为经验常数、仅对$n_e$实施单参数反演,既有物理依据(二阶微扰),也有统计支撑(后验变异系数),是一种适用于工程实施的降维策略。第五章将在色散函数数学形式已知、真值可控的等效验证载体上,进一步检验该链路的可移植性。
