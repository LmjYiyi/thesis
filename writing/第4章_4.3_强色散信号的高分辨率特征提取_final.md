# 4.3 强色散信号的高分辨率特征提取

本节基于"全局非平稳、局部近似平稳"的物理特性，将强色散差频信号的时频问题分解为一系列局部窄带估计问题：在足够短的滑动窗口内，瞬时频率变化可近似视为缓慢，从而允许使用局部高分辨率频率估计方法，最终重构出频率—时延特征轨迹$^{[22-27,53,54]}$。


## 4.3.1 基于短时观测窗的时频解耦与局部信号线性化近似

滑动窗口方法基于局部平稳近似：差频信号虽在整个调制周期内呈现显著非平稳特性，但在远短于$T_m$的局部时间窗内仍可视为窄带信号。对于第$i$个窗口，其中心时刻$t_i$对应的探测频率$f_{probe}(t_i)=f_0+Kt_i$由系统参数确定，窗口内估计得到的局部差频则映射为测量时延，由此形成用于后续反演的离散频率—时延特征点。

短时观测窗的长度$T_w$需要在"局部平稳性"与"频率估计方差"之间折中。一方面，若窗口过长，窗内频率漂移$\delta f = |\alpha|T_w$过大，将破坏局部近似；另一方面，若窗口过短，则有效采样点数不足，频率估计方差显著上升。为此，本文采用"窗内频率漂移不超过目标分辨率一定比例"的准则：

$$\delta f = |\alpha|T_w \le \beta \cdot \Delta f_{target} \tag{4-12}$$

结合本文强色散场景下的典型参数量级，最终取$T_w=12\,\mu s$，并采用90%重叠率进行滑窗处理，以在局部平稳性、时间分辨率与边缘稳健性之间取得平衡。

在确定窗口时长$T_w$后，需要设计窗口的滑动策略以实现对全调制周期的连续覆盖。设窗口滑动步长为$T_{step}$，则在$[0,T_m]$范围内可划分出$N_w$个观测窗口。以本文仿真参数为例：$T_m=50\,\mu s$、$T_w=12\,\mu s$，采用90%重叠率即$T_{step}=1.2\,\mu s$，则理论窗口数量约为

$$N_w = \left\lfloor \frac{T_m - T_w}{T_{step}} \right\rfloor + 1 \approx 32$$

第$i$个窗口的中心时刻$t_i$对应的探测频率（即发射信号的瞬时频率）为：

$$f_{probe}(t_i) = f_0 + K \cdot t_i \tag{4-13}$$

$f_{probe}(t_i)$仅由雷达系统参数决定，与待测介质参数无关，因此可作为观测模型中先验已知的频率坐标。

在第$i$个窗口内，利用高分辨率频率估计算法（如ESPRIT）提取局部差频$f_{beat,i}$。根据LFMCW的混频原理，该差频与瞬时群时延$\tau_g(t_i)$满足线性关系$f_{beat,i} = K \cdot \tau_g(t_i)$，由此可直接计算得到测量时延：

$$\tau_{meas,i} = \frac{f_{beat,i}}{K} \tag{4-14}$$

该测量时延$\tau_{meas,i}$是基于LFMCW混频关系得到的表观时延，并不严格等同于对应探测频率处的真实群时延。因此，后续反演中仍需引入理论模型对其进行统一拟合与修正。

将式(4-13)与式(4-14)联立，每个滑动窗口输出一个特征点$(f_{probe,i}, \tau_{meas,i})$。全部$N_w$个窗口形成的特征点集$\mathcal{D}=\{(f_{probe,i},\tau_{meas,i})\}_{i=1}^{N_w}$，即构成了频率—时延特征轨迹的离散采样。该轨迹的整体形态由等离子体色散特性决定，并将作为后续参数反演的直接观测输入。

图4.4(a)展示了传统全频段FFT在强色散下的散焦失效。强色散使瞬时差频呈现显著的频率依赖性，全时域FFT将这种非平稳特性平均为一宽频散焦区域，差频与探测频率之间不再存在唯一映射，传统方法因此失效。图4.4(b)给出了滑动窗口处理结果，理论差频轨迹（蓝色虚线）与提取散点（橙色）高度一致，表明该方法能够在强色散条件下恢复差频随探测频率变化的连续轨迹。

图4.4的仿真参数设置如下：等离子体截止频率$f_p = 33$ GHz，碰撞频率$\nu_e = 1.5$ GHz，等离子体厚度$d = 0.15$ m；LFMCW雷达扫频范围34.2-37.4 GHz（带宽$B = 3.2$ GHz），扫频周期$T_m = 50$ $\mu$s，对应调频斜率$K = 64 \times 10^{12}$ Hz/s。滑动窗口参数：窗口时长$T_w = 12$ $\mu$s，窗口步长$T_{step} = 1.2$ $\mu$s（对应90%重叠率），边缘保护区间$[0.05T_m, 0.95T_m]$。在该配置下，有效窗口数约为32个，覆盖探测频率范围约34.4-37.2 GHz。

在调制周期首尾约$0.05 T_m$的边缘区域内，LFMCW信号存在过渡瞬态，差频稳定性较差，色散条件下这一问题会进一步放大并表现为ESPRIT估计跳变。因此，特征提取仅保留$t \in [0.05 T_m, 0.95 T_m]$范围内的窗口。与此同时，本文采用90%重叠率的高密度滑窗策略（$T_{step} = T_w/10$），以平滑边缘处的估计突变并提高后续反演的稳健性。

当探测频率接近截止频率$f_p$时，信号幅度急剧衰减（见4.2.1节图4.1(b)右轴的幅度曲线），信噪比严重恶化。在这些低信噪比区域，ESPRIT算法的频率估计方差显著增大，甚至可能完全失效。为了量化各特征点的可靠性，本文在每个窗口内同步提取信号的均方根（RMS）幅度$A_i$作为置信度权重：

$$A_i = \sqrt{\frac{1}{N_{sam}}\sum_{n=1}^{N_{sam}}|s_D(t_{i,n})|^2} \tag{4-15}$$

其中$N_{sam}$为窗口内的采样点数（注意区分$N_w$为总窗口数）。该幅度信息在后续加权似然函数中作为置信度权重：高幅度区域权重大，低幅度区域权重自动降低，从而抑制低信噪比散点对反演结果的污染。这一处理与前文给出的"幅度对$\nu_e$更敏感、时延对$n_e$更敏感"结论保持一致。


## 4.3.2 多径干扰下的信源数估计（MDL准则）与子空间净化

在实际的微波透射诊断系统中，除了穿过等离子体介质的直达路径外，电磁波还会在系统组件（天线馈口、波导壁、介质界面）发生多次反射，形成多径传播效应。这些反射波与直达波在接收端叠加，导致混频后的差频信号不再是单一频率成分，而是包含多个离散频率分量的复合信号。

从信号模型的角度，设直达波对应的差频为$f_D^{(0)}$，第$k$条反射路径对应的差频为$f_D^{(k)}$，则窗口内的差频信号可表示为：

$$s_D(t) = \sum_{k=0}^{K-1} A_k \exp\{j[2\pi f_D^{(k)} t + \phi_k]\} + n(t) \tag{4-16}$$

其中$A_k$和$\phi_k$分别为第$k$条路径的幅度和初相，$n(t)$为加性高斯白噪声，$K$为总信源数（包含直达波和反射波）。由于反射路径长于直达路径，反射波经历的群时延更大，对应的差频$f_D^{(k)} > f_D^{(0)}$（$k \ge 1$）。

若直接对式(4-16)描述的多分量信号进行FFT分析，频谱将呈现多峰结构，各峰对应不同路径的贡献。在峰值间距较大（即路径时延差显著）时，可通过峰值检测直接分离；然而，当路径时延差较小或信噪比较低时，频谱峰值发生重叠或模糊，传统方法难以准确提取直达波分量。更严重的是，若误将反射波峰值当作直达波处理，将导致系统性的时延高估，反演得到的电子密度严重偏高。

为在多径干扰下准确提取直达波频率，本文引入基于子空间分解的高分辨率频率估计方法。将窗口数据按Hankel结构组织后，构造协方差矩阵并进行前后向平均，以提升弱信号条件下的统计稳定性。子空间维度$L$取窗口样本数的一半，使Hankel矩阵在维度与快拍数之间保持相对均衡，从而兼顾频率分辨率与协方差估计稳定性。具体而言，Hankel数据矩阵$\mathbf{X}$的构造、前向协方差$\mathbf{R}_f = \frac{1}{M}\mathbf{X}\mathbf{X}^H$以及前后向平均$\mathbf{R}_x = \frac{1}{2}(\mathbf{R}_f + \mathbf{J}\mathbf{R}_f^*\mathbf{J})$（其中$\mathbf{J}$为反对角单位矩阵）均按标准子空间方法执行。对$\mathbf{R}_x$进行特征值分解后，特征值通常呈"信号特征值+噪声平台"的分离结构，为后续信号子空间与噪声子空间的划分提供基础。

子空间方法的有效性高度依赖于信源数$K$的准确估计。若$K$估计过小，部分弱反射波分量被误判为噪声而丢失；若$K$估计过大，噪声分量被误判为信号，导致后续频率估计出现伪峰。本文采用基于信息论的最小描述长度（Minimum Description Length, MDL）准则进行信源数估计$^{[56]}$，其代价函数定义为：

$$\text{MDL}(k) = -\ln\left(\frac{\prod_{j=k+1}^{L}\lambda_j^{\frac{1}{L-k}}}{\frac{1}{L-k}\sum_{j=k+1}^{L}\lambda_j}\right)^{(L-k)M} + \frac{1}{2}k(2L-k)\ln M \tag{4-17}$$

其中，$L$为子空间维度，$M = N_w - L + 1$为快拍数，$\lambda_j$为协方差矩阵的特征值。式(4-17)由拟合项与复杂度惩罚项组成，本质上是在拟合度与模型复杂度之间寻求平衡。本文采用的MDL准则并非一次性的全局信源数判决，而是嵌入滑动时频处理框架中逐窗执行。对每个分析窗口，均独立构造Hankel矩阵、估计协方差并计算MDL代价函数，据此得到该窗口的最优信源数估计$\hat{K}_i$。

信源数的最优估计$\hat{K}$通过最小化MDL代价函数获得：$\hat{K} = \arg\min_{0 \le k \le L-1} \text{MDL}(k)$。在此基础上，为进一步提升工程实现的鲁棒性，本文引入了基于物理先验的软约束机制：

$$K_{final} = \max(1, \min(\hat{K}, K_{max})) \tag{4-18}$$

其中$K_{max} = 3$，对应直达波加两条主要反射路径的典型场景。该约束并未破坏MDL的自适应性，而是划定了合理的物理搜索边界——在实际透射诊断系统中，超过3条有效传播路径的情况极为罕见。下限强制为1，确保至少提取一个频率分量。

图4.5验证了单个分析窗口内MDL准则的判决特性。图4.5(a)在$K_{true}=2$的仿真条件下，MDL代价函数在$k=2$处稳定取得全局极小值，说明该判据能够可靠锁定真实信源数。图4.5(b)进一步给出了MDL与AIC在低信噪比条件下的对比结果，MDL对过拟合模型的惩罚更强，因此本文采用MDL作为信源数估计判据。

在获得信源数估计$\hat{K}$后，可将协方差矩阵的特征向量按特征值大小分为信号子空间$\mathbf{U}_s = [\mathbf{u}_1, \cdots, \mathbf{u}_{\hat{K}}]$与噪声子空间$\mathbf{U}_n = [\mathbf{u}_{\hat{K}+1}, \cdots, \mathbf{u}_L]$。信号子空间$\mathbf{U}_s$张成的空间包含了全部$\hat{K}$个频率分量的信息，而噪声子空间$\mathbf{U}_n$与信号正交，仅包含白噪声的贡献。这种正交分解实现了子空间净化——通过将观测数据投影到信号子空间，有效滤除了噪声子空间的干扰。

然而，子空间净化只完成了信号与噪声的分离，尚未解决直达波与反射波的区分。后续ESPRIT算法将在信号子空间$\mathbf{U}_s$中估计全部$\hat{K}$个频率分量$\{f_D^{(k)}\}_{k=1}^{\hat{K}}$。本文据此采用最小时延优先、最大幅度辅助的判据识别直达波：由于直达路径光程最短，其群时延最小，相应差频也最低；若幅度信息与最小时延判据冲突，仍以最小时延为准。据此，直达波频率的提取策略可形式化为：

$$f_{D,direct} = \min\{f_D^{(k)} : f_D^{(k)} \in [f_{min}, f_{max}]\} \tag{4-19}$$

其中$f_{min}$和$f_{max}$为基于物理约束设定的频率有效范围（典型值$f_{min} = 50$ kHz，$f_{max} = 10$ MHz），用于剔除明显不合理的异常估计值。


## 4.3.3 基于TLS-ESPRIT的频率-时延特征轨迹高精度重构

在完成信源数估计与子空间分离后，本文采用全最小二乘旋转不变子空间技术（TLS-ESPRIT）从信号子空间中提取各频率分量。ESPRIT无需谱峰搜索，计算效率和参数估计精度均优于MUSIC算法。

ESPRIT利用信号子空间的旋转不变性：Hankel矩阵的上下子矩阵$\mathbf{X}_1 = \mathbf{X}(1{:}L{-}1,:)$与$\mathbf{X}_2 = \mathbf{X}(2{:}L,:)$满足

$$\mathbf{X}_2 = \mathbf{X}_1 \mathbf{\Phi} \tag{4-20}$$

其中$\mathbf{\Phi} = \text{diag}(e^{j2\pi f_D^{(1)}/f_s'}, \cdots, e^{j2\pi f_D^{(K)}/f_s'})$为对角旋转矩阵，对角元素直接包含待估频率。在有噪声条件下，从信号子空间$\mathbf{U}_s$的上下子矩阵$\mathbf{U}_{s1}$、$\mathbf{U}_{s2}$出发，TLS-ESPRIT承认两侧均受噪声污染，通过

$$\mathbf{\Psi} = (\mathbf{U}_{s1}^H\mathbf{U}_{s1})^{-1}\mathbf{U}_{s1}^H\mathbf{U}_{s2} \tag{4-21}$$

估计旋转矩阵，相比标准LS-ESPRIT在低信噪比下具有更高的频率估计精度。对$\mathbf{\Psi}$做特征值分解，各频率分量由特征值相位角直接给出：

$$f_D^{(k)} = \frac{f_s'}{2\pi}\angle z_k \tag{4-22}$$

式(4-22)将频率估计转化为矩阵特征值问题，避免了FFT的栅栏效应，实现超分辨率频率估计。

将上述算法应用于每个滑动窗口，即可获得一系列离散特征点$(f_{probe,i}, \tau_{meas,i}, A_i)$，并构成数据集：

$$\mathcal{D} = \{(f_{probe,i}, \tau_{meas,i}, A_i)\}_{i=1}^{N_w} \tag{4-23}$$

其中，$f_{probe,i}$由系统参数决定，作为反演的自变量基准；$\tau_{meas,i}$对应局部测量时延；$A_i$反映该频点的信噪比水平，为后续加权反演提供依据。

图4.6展示了典型强色散工况下的频率—时延轨迹重构结果。TLS-ESPRIT提取的特征散点整体紧贴理论Drude曲线，在接近截止频率的高非线性区仍能保持连续跟踪；相比之下，全周期FFT基线因频谱展宽与读峰离散化误差而出现明显偏离。数值结果表明，ESPRIT重构轨迹的RMSE为0.0037 ns，最大误差为0.0099 ns，显著优于全周期FFT基线。图中散点颜色表示幅度权重$A_i$：远离截止频率时权重较高，接近截止频率时因透射衰减增强而降低，这一设置也与后续加权似然中的置信度设计保持一致。


## 本节小结

本节围绕强色散差频信号的非平稳特性，建立了基于滑动窗口—MDL—TLS-ESPRIT的高分辨率特征提取框架。其核心思想是利用"全局非平稳、局部近似平稳"的物理特性，将全周期信号分解为一系列局部频率估计问题，并进一步重构频率—时延特征轨迹。针对实际链路中的多径与低信噪比影响，本文通过MDL信源数估计、子空间分离及幅度加权机制提升了特征提取的稳健性。最终得到的特征数据集$\mathcal{D}=\{(f_{probe,i},\tau_{meas,i},A_i)\}$为后续贝叶斯参数反演提供了直接观测输入。
