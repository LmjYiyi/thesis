# 5.2 微波带通滤波器的色散物理等效机理

5.1节的标定已经确认，LFMCW系统在非色散环境下具备皮秒级时延分辨能力。下一步的验证并非直接将硬件系统置于真实等离子体环境中，而是先引入一个色散函数数学形式已知、响应真值可稳定导出的受控载体。这可用于验证第五章的方法链路能否可靠提取时延轨迹特征，并说明这些特征在已知正向模型条件下足以支撑后续参数计算。

选择微波带通滤波器作为等效验证载体，主要在于其中心频率、绝对带宽和阶数等物理参数完全可控，适合构造不同梯度的色散场景。同时，器件的$S$参数与群时延可通过高保真网络仿真稳定导出，能为轨迹提取误差提供绝对的真值比对点。滤波器作为无源线性时不变网络，不受等离子体实验中放电波动与失稳的干扰，能够为验证信息提取与参数反演算法提供纯净的环境。

## 5.2.1 滤波器色散演化与等离子体截止谐振的物理同构性

色散介质中群时延急剧变化的根源，在于传播常数在某一临界频率附近发生快速转折。对Drude等离子体而言，第三章已经给出群时延与截止频率$f_p$的关系，如式(5-4)所示

$$\Delta\tau_g(f) = \frac{d}{c}\left(\frac{1}{\sqrt{1-(f_p/f)^2}} - 1\right), \quad f > f_p \tag{5-4}$$

当$f$从高频端逐渐逼近$f_p$时，分母趋于零，群时延快速增大。这说明等离子体色散的主要信息集中在截止频率附近，群时延曲线的拓扑位置由$f_p$直接控制。

带通滤波器虽然不具备等离子体那样的单边截止结构，但在通带上下边缘同样存在传输从可通过到被抑制的快速过渡。设中心频率为$F_0$、绝对带宽为$BW$，则下、上截止频率分别为$f_L = F_0 - BW/2$和$f_H = F_0 + BW/2$。在$f_L$和$f_H$附近，$S_{21}$幅度迅速衰减，相位也随之急剧变化，因此群时延在通带两侧形成明显峰值。就局部行为而言，滤波器边缘的群时延突变与等离子体在$f \to f_p^+$附近的渐近增长具有相同的观测意义：群时延都在特征频率附近出现最强响应。

两者的差异在于拓扑形式：Drude等离子体表现为单边渐近发散，而滤波器表现为双边峰值结构。但在等效验证层面，核心不在于曲线是否呈现全局同形，而在于差频信号能否如实映射由色散激发的显著时延轨迹，且该轨迹能否受少数关键参数稳定控制。在这两点要求上，滤波器与等离子体完全一致。滤波器的通带平坦区对应远离截止频率的弱色散区，边缘峰值区则直观模拟了强色散区，能够在同一器件内部署梯次递进的真实色散测试环境。

采用滤波器进行等效验证还有直接的工程优势。滤波器响应真值可由ADS或网络分析仪获取，用以精确评价特征提取误差；滤波器参数可控，便于开展系统性的敏感度分析。滤波器具备时间稳定性，使测量残差主要源自信号处理链路本身，而非介质状态的实时波动，从而为时延轨迹提取的可靠性验证提供纯净的环境底噪。

## 5.2.2 基于切比雪夫传递函数的群时延正向理论模型构建

为了检验提取出的时延散点是否具备足够的信息承载量，必须先建立与滤波器对应的正向群时延模型。本节选用切比雪夫Type-I带通滤波器作为等效模型，其优点是数学形式明确、参数含义清晰，且便于与ADS仿真保持一致。

N阶切比雪夫Type-I低通原型的幅度平方响应如式(5-5)所示

$$|H_{LP}(j\Omega)|^2 = \frac{1}{1 + \varepsilon^2 T_N^2(\Omega)} \tag{5-5}$$

其中，$\Omega$为归一化角频率，$T_N(\Omega)$为N阶切比雪夫多项式，$\varepsilon = \sqrt{10^{R_p/10} - 1}$为纹波因子，$R_p$为通带纹波深度。将低通原型映射到带通结构后，归一化频率变量写为式(5-6)

$$\Omega = \frac{1}{BW_n}\left(\frac{\omega}{\omega_0} - \frac{\omega_0}{\omega}\right) \tag{5-6}$$

其中，$\omega_0 = 2\pi F_0$，$BW_n = BW/F_0$。该频率映射方式将低通特性的单边滚降低频衰落模型，数学平移为吻合射频带通滤波特征的双向衰减约束通道。

在复$s$域中，中心频率处的N阶切比雪夫带通滤波器传递函数通常表示为：

$$H_{BP}(s) = \frac{b_0 + b_1 s + \cdots + b_{2N} s^{2N}}{a_0 + a_1 s + \cdots + a_{2N} s^{2N}} \tag{5-7}$$

其中，多项式系数$\{b_k\}$和$\{a_k\}$由$(F_0, BW, N, R_p)$唯一确定。对本章采用的5阶滤波器，$F_0 = 37$ GHz，$BW = 1$ GHz，$N = 5$，$R_p = 0.5$ dB，对应10阶有理函数。

群时延在网络理论中定义为传输网络连续相位响应对角频率的微商负值，计算基准如式(5-8)所示

$$\tau_g(\omega) = -\frac{d\phi(\omega)}{d\omega} \tag{5-8}$$

其中传输相位的折叠与展开关系可由通带复频率响应函数求取，如式(5-9)所示

$$\phi(\omega) = \arg\left[H_{BP}(j\omega)\right] = \mathrm{unwrap}\left\{\arctan\frac{\mathrm{Im}[H_{BP}(j\omega)]}{\mathrm{Re}[H_{BP}(j\omega)]}\right\} \tag{5-9}$$

由于高阶有理复多项式的相位微分形式难以推演闭合显式解析表达，工程测算普遍引入离散近似中心差分网络实现群时延推演计算：

$$\tau_g(\omega_k) \approx -\frac{\phi(\omega_{k+1}) - \phi(\omega_{k-1})}{\omega_{k+1} - \omega_{k-1}} \tag{5-10}$$

因此，正向模型的计算链路可以概括为：给定$(F_0, BW, N, R_p)$，先生成传递函数系数，再计算复频率响应、展开相位，最后通过数值微分得到群时延曲线。这一过程与MATLAB中`cheby1`和`freqs`的调用逻辑一致，也便于与5.3节的拟合程序和ADS仿真保持同一物理口径。

在标称参数$(F_0 = 37$ GHz，$BW = 1$ GHz，$N = 5$，$R_p = 0.5$ dB$)$下，模型计算得到典型的双峰群时延结构。通带下边缘和上边缘各出现一个峰值，峰高约为6~7 ns，通带中央基线约为2 ns。双峰之间的间距主要由$BW$决定，峰位由$F_0$决定，峰形陡峭程度则主要受$N$和$R_p$影响。该模型已满足本章等效验证的全部需求：它稳定输出由少数参量约束的色散理论真值，为评估LFMCW提取的时延散点是否具备足够的物理信息载量提供了唯一的绝对基准。

## 5.2.3 前向物理模型参数敏感度分析

在建立正向模型后，还需明确各参数对群时延曲线的控制方式。该分析用于解释后续拟合结果，同时说明滤波器作为等效载体的可行性。若某个参数对曲线拓扑位置、宽度和尖锐度的影响彼此可区分，则说明提取出的时延轨迹具备驱动后续参数计算的可能性。

当中心频率$F_0$在$\pm 500$ MHz范围内变化时，双峰结构基本保持形状不变，仅在频率轴上整体平移。峰高、峰宽和峰间距变化很小，因此$F_0$主要控制色散结构的绝对位置。这一点与等离子体中截止频率对群时延曲线位置的控制方式一致，都是一阶主导参数。

当绝对带宽$BW$从0.8 GHz变化到1.2 GHz时，双峰间距随之拉伸或压缩，峰值高度也发生明显变化。带宽减小时，品质因数增大，边缘相位过渡更陡，群时延峰值升高；带宽增大时，双峰被拉开而峰值下降。因此，$BW$主要控制曲线的几何尺度和色散强度，敏感度次于$F_0$，但仍明显高于弱微扰参数。

对于等效阶数$N$，其主要影响边缘相位陡峭度。当$N$从3增加到7时，双峰边缘变得更陡，通带内纹波也更明显。与$F_0$和$BW$相比，$N$对峰位和峰间距的影响较弱，主要改变局部细节和通带内微结构。这意味着在有限噪声和有限采样密度下，$N$的可观测性通常低于$F_0$和$BW$。

由此可以得到清晰的敏感度层级：$F_0 > BW > N$。其中，$F_0$控制拓扑位置，$BW$控制尺度和峰高，$N$控制边缘陡峭度和细部纹波。该层级结构与前文对等离子体参数敏感度的分析在机制上是一致的：决定色散主拓扑的参数最容易被群时延观测通道识别，而控制损耗或细节结构的参数则需要更高维度的信息支撑。后续的模型拟合检查，正是要验证LFMCW系统提取的时延散点能否有效恢复这一主导层级的物理特征。
