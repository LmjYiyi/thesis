# 3.4 色散效应对差频信号的调制机理与误差解析

前两节分析了群时延的频域非线性特征,但尚未回答:在LFMCW的动态扫频工作模式下,色散如何具体影响差频信号的波形与频谱?本节基于局部泰勒展开,推导色散条件下差频信号的时域与频域表达式,定量分析二阶色散导致的频谱散焦效应与带宽耦合机制。

## 3.4.1 群时延的二阶泰勒级数展开与时变时延模型

LFMCW发射信号的瞬时角频率$\omega(t) = \omega_0 + K' t$,其中$K' = 2\pi B/T_m$为角频率调制斜率。将总群时延$\tau(\omega)$在扫频起始角频率$\omega_0$处作二阶展开:

$$\tau(\omega) = \tau_{g,0} + \tau_1[\omega-\omega_0] + \frac{1}{2}\tau_2[\omega-\omega_0]^2 + \cdots \tag{3-14}$$

其中$\tau_{g,0} = \tau(\omega_0)$为扫频起点处的总群时延(区别于真空基底时延$\tau_0=d/c$),$\tau_1 = d\tau/d\omega|_{\omega_0}$,$\tau_2 = d^2\tau/d\omega^2|_{\omega_0}$。该截断在分析窗口内频偏$|K't|$足够小且工作点与截止奇异点保持裕量时成立。代入$\omega(t)$得时变时延:

$$\tau(t) = A_0 + A_1 t + A_2 t^2 \tag{3-15}$$

其中$A_0 = \tau_{g,0}$,$A_1 = \tau_1 K'$,$A_2 = \tau_2 (K')^2/2$。$A_2 \propto B^2/T_m^2$说明带宽会显著放大二阶色散效应。

由式(3-9)的物理模型,通过链式法则可得显式表达:

$$\tau_1 = \frac{1}{2\pi} \cdot \left(-\frac{\tau_0}{f_0} \cdot \frac{(f_p/f_0)^2}{[1-(f_p/f_0)^2]^{3/2}}\right) \tag{3-16}$$

$$\tau_2 = \frac{1}{(2\pi)^2} \cdot \frac{3\tau_0}{f_0^2} \cdot \frac{(f_p/f_0)^2}{[1-(f_p/f_0)^2]^{5/2}} \tag{3-17}$$

$\tau_2$在截止频率附近呈$[1-(f_p/f_0)^2]^{-5/2}$的奇异性增长,这是二阶色散在强色散区显著的根源。以Ka波段典型工况验算,半带宽内二阶项贡献约27.42 ps,三阶余项约6.99 ps,二阶模型适合用于解释差频相位畸变和频谱散焦的形成机理。

## 3.4.2 差频信号相位的非线性畸变与瞬时频率解析

在3.4.1节时变时延模型与慢变时延近似下,混频后差频相位整理为:

$$\Delta\phi(t) = \omega_0 \tau(t) + \pi\frac{B}{T_m}\left\{2t\,\tau(t) - \tau^2(t)\right\} \tag{3-18}$$

将式(3-15)代入并忽略$t^3$及以上高阶项:

$$\Delta\phi(t) = \phi_0 + L t + Q t^2 \tag{3-19}$$

差频中心频率$f_D'$满足$2\pi f_D' = L$:

$$f_D' = K\tau_{g,0} + K\tau_1(\omega_0-K'\tau_{g,0}) \tag{3-20}$$

线性调频率$\alpha = Q/\pi$:

$$\alpha = \frac{\omega_0 \tau_2 (K')^2}{2\pi} + \frac{2B}{T_m}\tau_1 K' - \frac{B}{T_m}(\tau_1 K')^2 - \frac{B}{T_m}\tau_{g,0}\tau_2 (K')^2 \tag{3-21}$$

瞬时频率为:

$$f_D(t) = f_D' + \alpha t \tag{3-22}$$

式(3-22)表明差频信号已从稳态单频形式转变为瞬时频率随时间线性漂移的非稳态调频信号。

## 3.4.3 频谱特征量化:二阶色散导致的散焦效应与带宽耦合机制

瞬时频率在$[0, T_m]$内的扫频范围定义频谱展宽:

$$\Delta f_D = |\alpha| T_m \tag{3-23}$$

展宽正比于二阶色散系数$\alpha$与扫频周期$T_m$的乘积。将$\alpha$改写为$B$的函数并代入:

$$\Delta f_D = \left|2\pi\frac{B^2}{T_m}(\omega_0\tau_2+2\tau_1) - 4\pi^2\frac{B^3}{T_m^2}(\tau_1^2+\tau_{g,0}\tau_2)\right| \tag{3-24}$$

当$B$较小时三次项可忽略:

$$\Delta f_D \approx 2\pi\frac{B^2}{T_m}|\omega_0\tau_2+2\tau_1| \tag{3-25}$$

频谱展宽对带宽近似呈二次关系$\Delta f_D \propto B^2$:增大带宽能提高分辨率,却平方倍地加剧散焦;$f_p$接近$f_0$时$\tau_2$的急剧增大进一步放大效应。式(3-24)和(3-25)描述展开点附近的局部增长律,不宜直接外推为全局定律。

![图3-10 不同色散强度下差频信号的FFT频谱特征](figures/final_output_doc/fig3.10.tiff)
