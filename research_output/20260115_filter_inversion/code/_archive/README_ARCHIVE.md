# 代码归档说明

本目录存放滤波器参数反演研究的**历史版本代码**。

---

## 📜 版本历史

### v0: 可行性验证版 (最早版本)

**文件**：`filter_inversion_feasibility.m`  
**日期**：2026-01-15 (最初)  
**功能**：
- **双参数反演**：F0（中心频率）、B（带宽）
- 简单的能量加权策略
- 敏感性分析
- 👉 **这是整个研究的起点**，证明了算法可行性

**亮点**：
- 清晰的理论推导注释
- 完整的可视化（残差、权重、敏感性曲线）
- 成功验证：F0精度<1%, B精度<5%

**局限**：
- 未包含阶数N反演
- 权重策略较简单
- 未优化收敛性能

---

### v1: 双参数优化版

**文件**：`filter_inversion_LM.m`  
**日期**：2026-01-15  
**改进**：
- 参考LM.m的能量加权 `(W/max(W))^2`
- 参数归一化 `scale=1e10`
- 更鲁棒的物理约束

---

### v2: 三参数初版

**文件**：`filter_inversion_3param_LM.m`  
**日期**：2026-01-15  
**改进**：
- **新增阶数N反演**
- 边缘增强权重（解耦B和N）
- 更严格的参数约束

**问题**：
- B和N误差较大（14% ~ 12%）
- 需要进一步优化

---

### v3: LFMCW方法探索（失败版）

**文件**：`filter_inversion_LFMCW.m`  
**日期**：2026-01-15  
**尝试**：
- 完整LFMCW信号处理链
- ESPRIT算法提取群时延
- 混频、降采样、滑动窗口

**失败原因**：
- 滤波器相位建模不准确：`phi = -omega * tau_g` (错误!)
- 导致ESPRIT测量偏差巨大
- 反演失败：BW误差>50%

---

### v4: 简化验证版

**文件**：`verify_inversion_algorithm.m`  
**日期**：2026-01-15  
**目的**：
- 绕过LFMCW，直接用理论数据验证算法正确性
- 证明：反演算法本身是准确的

---

## ✅ 当前推荐版本

**请使用主目录中的**：
- **`verify_inversion_optimized.m`** (理论验证)
- **`LFMCW_filter_inversion_FINAL.m`** (工程级LFMCW)

这两个文件整合了所有优化，达到了最高精度！
