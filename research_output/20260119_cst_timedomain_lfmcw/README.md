# CST时域仿真与LFMCW信号集成研究

**研究日期**：2026-01-19
**研究目标**：通过MATLAB控制CST进行时域仿真，实现LFMCW激励信号注入

## 文件清单
- `report.md` - 完整技术报告
- `code/generate_lfmcw_excitation.m` - LFMCW激励信号生成器

## 研究结论

### 可行性评估
| 方面 | 评估 | 备注 |
|------|------|------|
| CST支持 | ✅ 可行 | 需要时域求解器许可 |
| MATLAB集成 | ⚠️ 部分 | COM接口可能受限 |
| 计算成本 | ⚠️ 较高 | 需优化参数 |

### 推荐方案
采用**混合方案**：
1. MATLAB生成激励信号文件
2. CST GUI手动触发时域仿真
3. MATLAB读取输出信号进行后处理

## 后续工作
1. 在CST GUI中手动验证时域仿真可行性
2. 确认用户自定义激励信号格式
3. 实现MATLAB自动化集成
